# Shinhan Bank IT Security Policy

## Password Requirements

All employees must follow these password guidelines:
- Minimum 12 characters
- Must include: uppercase, lowercase, number, and special character
- Password change required every 90 days
- Cannot reuse the last 10 passwords
- Account locks after 5 failed attempts

## Device Management

### Company Devices
- All company laptops must have full disk encryption enabled
- Auto-lock screen after 5 minutes of inactivity
- USB ports are disabled by default on all workstations
- Software installation requires IT department approval

### Personal Devices (BYOD)
- Personal devices are NOT allowed to access internal banking systems
- Company email can be accessed on personal phones via Shinhan Mobile MDM only
- Employees must register personal devices with IT Security team

## Data Classification

| Level | Description | Examples |
|-------|-------------|----------|
| Confidential | Highest sensitivity | Customer financial data, credentials |
| Internal | Business sensitive | Strategy documents, financial reports |
| Restricted | Limited access | HR records, audit reports |
| Public | Open information | Marketing materials, press releases |

## Incident Reporting

All security incidents must be reported within 1 hour to:
- IT Security Hotline: ext. 5555
- Email: security@shinhan.com
- Shinhan Security Portal: https://security.shinhan.internal

## Remote Access

- VPN is required for all remote access to internal systems
- Two-factor authentication (2FA) is mandatory
- Remote sessions timeout after 30 minutes of inactivity
- Screen sharing during remote sessions must be authorized
